Bạn là một senior product designer và frontend architect.

NHIỆM VỤ:
Thiết kế UI/UX tổng thể cho một web app "AI Study Assistant".

MỤC ĐÍCH APP:
App giúp sinh viên:
1. Nhập mục tiêu học tập
2. Sinh lộ trình học (roadmap)
3. Upload và phân tích tài liệu
4. Hỏi đáp thông minh (AI Tutor)
5. Tạo quiz / flashcard
6. Đánh giá năng lực
7. Theo dõi tiến độ học tập

YÊU CẦU:
- KHÔNG viết code
- Chỉ thiết kế cấu trúc UI toàn app
- Đề xuất:
  - Các màn hình chính (screens/pages)
  - Navigation (sidebar, topbar, menu)
  - Cách nhóm các chức năng hợp lý
  - Trang nào là trang chính (homepage)
  - Flow người dùng từ lần đầu vào app

UI STYLE:
- Hiện đại, tối giản
- Dễ dùng cho sinh viên
- Responsive
- Bố cục rõ ràng

OUTPUT:
Trả lời theo cấu trúc:
1. App structure (danh sách màn hình)
2. Navigation
3. Screen descriptions
4. User flow
5. UI style suggestions

QUAN TRỌNG:
- Không viết backend
- Không code
- Chỉ tập trung vào UI/UX architecture
